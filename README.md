# Tic-tac-toe

A simple tic-tac-toe game implementing a Minimax algorithm

# Dependency

Install rust 

```
$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

# Clone

Clone the project

```
$ git clone git@github.com:eclausse/tic-tac-toe.git
$ cd tic-tac-toe
```

# Play

```
$ cargo run
```
